# template-vue

A project template for Vue.js applications

## Features

- Vue 2.0, Vuex 2.0 and vue-router 2.0
- Testing using Testem, Mocha, power-assert and Sinon.JS
- Hot module replacement for Vue components and Vuex store

## Commands

```sh
# dev server
$ npm run dev

# test in development
$ npm run test

# test for CI
$ npm run test:ci

# release build
$ npm run release
```
